Jeroo's History

Early 1970's -- LOGO Turtles

  • Dr. Seymour papert of MIT had elementary students writing programs to control the movements of turtle robots.

Late 1970's -- Karel The Robot

  • Dr. Richard Pattis developed Karel the Robot as a learning tool. Students write program to move Karel around his virtual world, and have him interact with his environment.

1990 -- Jessica

  • Lai Kwan Tong's master's project at Illinois State University was Jessica, a derivative of Karel. Jessica introduced the metaphor and basic GUI layout that is used in Jeroo. Jessica was used for several years at Illinois State.

1999-2001 -- The Snipe Project

  • Student project teams at Northwest Missouri State University specifed and developed a full prototype for a java-based successor to Jessica. Students: Erman Ayvaz, Time Bell, Brian Dorn, Brett Graves, John Otte, Michael Parneley, Jolaine Zweifel

2002 -- Jeroo

  • Using the Snipe project as a starting point, Dean Sanders and Brian Dorn specified, designed, and implemented the first version of Jeroo. They are coninually revising and updating Jeroo to meet the needs of the growing community of users.

2019 -- Jeroo Web

  • Using the Jeroo project from 2002. Ben Konz, Calean Bryan, Thomas Connole, and John Adam have implemented the first version of a web based Jeroo. Ocaml was used to create the compiler. Angular was used for the web framework. Items added to the jeroo program are syntax highlighting, and jeroo board size editablity.
