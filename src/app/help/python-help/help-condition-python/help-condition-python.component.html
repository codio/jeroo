Conditions (Python Style)

A condition is just a Boolean expression -- an expression that is either true or false.
  • Conditions are used to define selection and repetition control structures
  • In Jeroo, a Boolean expression is any valid combination of Boolean operators and invocations of Boolean methods.
  • Therea re two Boolean constants: True and False
  • There are three Boolean operators: not, and, or.
    .
  • In the absence of parentheses to indicate otherwise, every not operator will be evaluated first, every and operator will be evaluated second, and every or operator will be evaluated third.


Operator: not
Suppose that P represents a Boolean expression.
The expression not P has the opposite truth value. 

Code example:
   not Kim.isWater(LEFT)
Operator: and
Suppose that P and Q represent Boolean expressions.
The expression P and Q is only true when both P and Q are true. 

Code example:
   Kim.hasFlower() and Kim.isNet(AHEAD)
Operator: or
Suppose that P and Q represent Boolean expressions.
The expression P or Q is only false when both P and Q are false. 

Code example:
   Pat.isJeroo(LEFT) or Pat.isJeroo(RIGHT)
